# Ctrlmoney
Ctrlmoney — это ультралегкое мобильное веб-приложение для мгновенного контроля, оцифровки и валидации списаний продуктов на кухнях ресторанов и кафе. Продукт решает проблему скрытых операционных потерь и воровства в сфере HoReCa, заменяя хаотичные бумажные журналы жестким цифровым регламентом.
